# Contributing to AI_RCD

Thank you for your interest in contributing to AI_RCD! We welcome contributions of all kinds, from bug reports to feature requests and code contributions.

## How to Contribute

### 1. Reporting Bugs
- Use the **Bug Report** issue template.
- Provide a clear description of the issue and steps to reproduce.
- Include environment details (OS, Python version, Docker version).

### 2. Suggesting Enhancements
- Use the **Feature Request** issue template.
- Describe the proposed change and why it would be beneficial.

### 3. Pull Requests
- Fork the repository and create a feature branch.
- Ensure your code follows the project's style (PEP 8 for Python).
- Run existing tests before submitting: `python run_all_tests.py`.
- Add new tests for your changes where appropriate.
- Update documentation if you've added or changed features.

## Development Setup

1. Clone your fork: `git clone https://github.com/your-username/AI_RCD.git`
2. Follow the setup instructions in the [README.md](README.md).
3. Install development dependencies:
   ```bash
   pip install -r requirements/development.txt  # If available
   ```

## Code of Conduct
This project adheres to the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
